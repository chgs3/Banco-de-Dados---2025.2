# Sistema de Gerenciamento Penitenciário

Projeto desenvolvido para a disciplina de Banco de Dados. O objetivo é gerenciar o fluxo de detentos, celas, visitas e sentenças de um complexo penitenciário.

## Integrantes
* Carlos Henrique

## Descrição do Minimundo
O sistema gerencia um complexo composto por diversos presídios. O foco principal é o controle da lotação das celas, histórico de crimes (sentenças) e registro de visitantes.
*Regra principal:* O sistema calcula automaticamente a data de saída baseada na soma das penas, respeitando o teto máximo de 30 anos (Requisito do projeto).

## Tecnologias Utilizadas
* **Banco de Dados:** Oracle (Compatível com PL/SQL)
* **Ferramenta:** Oracle Live SQL / brModelo
* **Modelagem:** Diagrama Entidade-Relacionamento (DER)
